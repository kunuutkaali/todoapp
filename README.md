# todoapp
Express mongoose project

# Setup to dev
1. Clone code
2. Setup .env
3. Run <code>npm install</code>
# .env
```env
 PORT=DESIREDPORT
 MONGODB_URI='DESIREDMONGODB_URI'
 ACCESS_TOKEN_SECRET=DESIREDSECRET
 REFRESH_TOKEN_SECRET=DESIREDSECRET
```
# Creating a server
2. Start a server using command:<code>npm start</code>
